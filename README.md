# 스테레오 이벤트 카메라 · FPGA 고정 브라켓 3D 프린팅 기록

스테레오 이벤트 카메라 모듈 2개와 FPGA 보드를 안정적으로 고정하기 위해 직접 설계하고 출력한 모듈형 브라켓의 제작 기록입니다. 3D 프린터를 처음 사용하는 사람이 실제 보드 고정용 부품을 설계·출력할 때 필요한 순서와 확인 항목을 함께 정리했습니다.

> 이 저장소의 STL은 특정 스테레오 이벤트 카메라 모듈과 FPGA 보드 조합을 기준으로 제작한 예시입니다. 다른 보드에 적용할 때는 나사 구멍 위치, 보드 두께, 커넥터 및 케이블 여유 공간을 반드시 다시 측정하세요.

## 출력물 개요

![삼각대에 결합한 스테레오 이벤트 카메라와 FPGA 고정 브라켓](assets/assembled-stereo-rig.jpg)

출력물은 크게 세 부분으로 나뉩니다.

1. **카메라 홀더 2개**: 각 이벤트 카메라 모듈의 아래쪽을 레일로 받치고, 나사 보스로 위치를 고정합니다.
2. **FPGA 보드 베이스**: FPGA 보드를 지지하고, 앞쪽의 반복 결합 구멍으로 카메라 홀더 위치를 조절합니다.
3. **결합부와 바닥 고정부**: 카메라 간 거리와 삼각대·장착판 결합을 위한 공용 체결 위치입니다.

![조립된 브라켓의 후면과 FFC 케이블 여유 공간](assets/assembled-rig-rear.jpg)

카메라 모듈의 렌즈 중심 간 거리가 스테레오 기준 거리이므로, 보드 외곽이 아니라 렌즈 중심을 기준으로 카메라 위치와 높이를 확인했습니다. FFC 케이블은 단순 선이 아니라 폭과 굽힘 반지름이 있는 부품으로 보고, 브라켓이 케이블 출구와 커넥터를 막지 않도록 공간을 남겼습니다.

## 3D 프린팅을 시작하기 전에

PCB 고정용 브라켓은 단순히 보드의 가로·세로 크기만 맞추는 작업이 아닙니다. CAD를 시작하기 전에 실제 부품을 앞에 두고 아래 항목을 측정했습니다.

- PCB 외곽 크기와 두께
- 나사 구멍의 **중심 좌표·중심 간격·직경**
- 렌즈 중심 또는 기준축 위치
- USB, 전원, FFC 등 커넥터의 위치와 높이
- 케이블이 빠져나가는 방향과 굽힘 공간
- 나사 머리, 너트, 드라이버가 들어갈 조립 공간

나사 구멍은 외곽 거리보다 중심 간격이 중요합니다. 외곽 간격만 측정했다면 `중심 간격 = 외곽 간격 + 구멍 지름`으로 환산해야 합니다.

## 이번 브라켓의 설계 방식

### 1. 실제 부품은 단순 모델로 먼저 만듭니다

실물 외관 전체를 복잡하게 모델링할 필요는 없습니다. PCB 외곽과 두께, 나사 구멍, 렌즈 중심, 커넥터·케이블의 최대 점유 공간만 정확하게 만든 뒤 브라켓과 함께 조립해 봤습니다. 특히 FFC는 통과 선 하나가 아니라 폭과 굽힘 반지름을 가진 `keepout` 공간으로 고려했습니다.

### 2. 접촉하는 곳만 잡습니다

보드 전체를 큰 판으로 덮는 대신 다음 구조만 남겼습니다.

- 보드 아래를 받는 하단 레일
- 나사 구멍을 잡는 보스
- 긴 구조의 처짐을 막는 얇은 리브
- 너트와 드라이버, 케이블을 위한 빈 공간

이 방식은 재료와 출력 시간을 줄이고, 포트 접근성을 유지합니다. 카메라 홀더와 FPGA 보드 베이스를 분리했기 때문에 카메라 종류가 바뀌면 홀더만 새로 설계해 교체할 수 있습니다.

### 3. 작은 시험편부터 출력합니다

전체 브라켓 전에 목표 구멍 지름을 0.1~0.2 mm 간격으로 바꾼 나사 구멍, 보스 하나, 짧은 레일 조각만 포함한 시험편을 먼저 출력하는 것을 권장합니다. CAD 구멍과 실제 출력 구멍은 프린터 상태와 필라멘트 수축 때문에 달라질 수 있습니다.

## 나사와 체결

![나사 용도 구분: 1번 모듈 고정, 2번 카메라 모듈 고정, 3번 FPGA 보드 고정](assets/fasteners.jpg)

| 사진 번호 | 사용 위치 | 확인 방법 |
| --- | --- | --- |
| 1 | 가장 큰 모듈·베이스 고정 나사 | 조립된 전체 모듈에서 하나를 빼서 길이와 나사산을 확인합니다. |
| 2 | 이벤트 카메라 모듈 고정 나사 | 카메라 PCB 구멍에 실제로 맞는 나사와 너트를 함께 시험합니다. |
| 3 | FPGA 보드 고정 나사 | FPGA 보드의 고정 구멍과 포트·FFC 간섭을 함께 확인합니다. |

회사에 보관된 나사가 묶음으로 있을 때는 규격 이름만 보고 고르지 않는 편이 안전합니다. **조립돼 있는 부품에서 나사 하나를 먼저 빼서, 같은 머리 모양·직경·길이의 나사를 찾아 사용하세요.** 나사산이 맞아도 길이가 너무 길면 PCB 뒤쪽 부품, 케이블, 너트 공간과 간섭할 수 있습니다.

이 설계에서 사용한 시작값은 카메라 PCB 관통 구멍 3.15 mm, FPGA 보드 관통 구멍 2.45 mm, 홀더·베이스 및 바닥 고정용 5.00 mm 계열 구멍입니다. 이는 특정 출력물에서 검증한 CAD 시작값일 뿐이므로, 다른 프린터나 다른 체결재에서는 반드시 시험편으로 조정해야 합니다.

## 출력 설정과 절차

### 사용 프로그램

- **FreeCAD**: mm 기준의 보드 더미 모델, 레일, 보스, 리브, 간섭 공간 설계
- **3DWOX Desktop / Slicer**: STL의 출력 방향, 서포트, raft, 레이어 설정 및 G-code 생성

![3DWOX Desktop에서 STL 출력 배치를 확인하는 예시](assets/slicer-layout.png)

### PLA 시작 설정 예시

아래 설정은 0.4 mm 노즐·PLA 브라켓 기준의 출발점입니다. 출력기와 필라멘트 상태에 맞춰 조정하세요.

| 설정 | 시작값 |
| --- | --- |
| 노즐 / 베드 온도 | 200 C / 60 C |
| 레이어 높이 | 0.20 mm |
| 벽·상하단 두께 | 0.80 mm 이상 |
| 인필 | 15%, 첫 실사용 부품은 20% 권장 |
| 일반 출력 속도 | 40 mm/s |
| 초기 레이어 속도 | 20 mm/s |
| 서포트 | 켬, 우선 `Touching buildplate` |
| 서포트 밀도 / XY 간격 / Z 간격 | 20% / 0.80 mm / 0.20 mm |
| 베드 부착 | Raft, 여유 5.00 mm |

### 출력 순서

1. STL을 슬라이서에 불러와 프린터와 재료 프로파일을 선택합니다.
2. 베드 크기와 높이 제한 안에 들어가는지 확인합니다.
3. 레일, 보스, 지붕 아래가 공중에 뜨지 않도록 방향과 서포트를 설정합니다.
4. 레이어 미리보기에서 나사 구멍, 너트 공간, FFC 창이 막히지 않는지 확인합니다.
5. 해당 프린터·재료 프로파일로 생성한 G-code만 사용합니다.
6. 작은 시험편부터 출력하고 실제 PCB·나사로 체결한 뒤 전체 세트를 출력합니다.

STL은 형상 데이터만 담고 있습니다. 출력 전에는 반드시 사용하는 프린터와 재료 프로파일로 슬라이스해 G-code를 생성해야 합니다. 다른 프린터용 G-code를 그대로 사용하면 시작 명령, 온도 제어, 베드 크기, 펌웨어 명령이 달라 문제가 생길 수 있습니다.

## 조립 전 점검표

- [ ] PCB 나사 구멍 중심과 브라켓 보스 중심이 맞는가
- [ ] 나사가 보스를 완전히 지나가고, 뒤쪽에 너트·와셔·드라이버 공간이 있는가
- [ ] 보드 바닥이 레일 바닥에 실제로 닿는가
- [ ] 레일과 리브가 커넥터, 납땜부, 렌즈 하우징을 누르지 않는가
- [ ] FFC·USB·전원 케이블의 삽입과 굽힘 방향이 열려 있는가
- [ ] 두 카메라의 렌즈 중심이 같은 높이와 방향에 있는가
- [ ] 카메라 간 거리를 렌즈 중심 기준으로 확인했는가
- [ ] 슬라이서 미리보기에서 구멍과 케이블 창이 서포트·모델 재료로 막히지 않는가
